# SOC Incident Investigation Portfolio

This repository contains a collection of simulated Security Operations Center (SOC) investigations performed in lab environments such as LetsDefend and TryHackMe.

## Included Investigations

- RDP Brute Force Attack
- PowerShell Malware Execution & Privilege Escalation
- SQL Injection Attack

## Skills Demonstrated

- Alert triage and validation
- Log correlation and analysis
- Threat intelligence enrichment
- Attack chain reconstruction
- Incident reporting and documentation

## Tools Used

- Splunk (log analysis)
- VirusTotal (threat intelligence)
- Wireshark (network analysis)

---

These investigations are structured to simulate real-world SOC workflows and demonstrate practical incident response skills.
