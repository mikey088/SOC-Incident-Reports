# SOC Detection Lab (Atomic Red Team + Splunk)

This project demonstrates hands-on experience simulating real-world attacks using Atomic Red Team and analysing logs in Splunk.

## 🔧 Lab Setup
- Attacker: Atomic Red Team (Windows VM)
- SIEM: Splunk (host machine)
- Logs: Windows Event Logs

## 🧪 Simulated Attacks
- T1059.001 – PowerShell Execution
- T1547.001 – Registry Persistence

## 🔍 Key Skills Demonstrated
- Log analysis using Splunk
- Detection of encoded PowerShell commands
- Process chain investigation (PowerShell → CMD → child processes)
- Identification of persistence mechanisms (Registry + Startup folder)
- MITRE ATT&CK mapping

Detailed investigation reports are included in this repository.
